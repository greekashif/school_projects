
To compile the implemenation file:
	make Program1Math.o  

To compile the main program file:
	make Program1.o

To link the object files:
	make Program1.exe

To compile and link:
	make compile  

To run:
	make run

To clean: 
        make clean (remove object files and .exe) 
	make cleano (remove object files)
	make cleanx (remove .exe)
